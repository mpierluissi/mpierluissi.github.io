The Universal Category System (UCS) Filename Assistant is a program for Windows created using the AutoHotkey scripting language. This project was inspired by Kai Paquin and his original UCS program written for KeyboardMaestro.

This program allows users to name audio files using the UCS format.

For more information about UCS, please visit www.universalcategorysystem.com

A tutorial video about Filename Assistant can be found here.

Please report any bugs by email using the contact page or make a post on the FnAsst Facebook page.
